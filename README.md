# se-login-signup

ICSI 418Y — Homework 1

This is a login page and a signup page. It was created as a class assignment and kept intentionally simple so the markup and CSS are easy to read and modify.

Project structure

```
se-login-signup/
  login.html
  signup.html
  styles.css
  README.md
```

What I implemented

- `login.html` — a straightforward login form (User ID, Password) with a link to the signup page.
- `signup.html` — a simple signup form (First name, Last name, User ID, Password) that links back to login.
- `styles.css` — a small stylesheet to keep the pages looking consistent and centered. No external frameworks used.

Preview locally

- Open `login.html` or `signup.html` in your browser (double-click), or use the Live Server extension in VS Code.

Commits and pushing

I made separate commits for the HTML, the CSS, and this README so the project history is clear. 

```powershell
git branch -M main
git remote add origin https://github.com/orenh57/se-login-signup.git
git push -u origin main
```
